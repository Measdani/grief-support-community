# Together in Grief - Support Community

A compassionate web platform where people who have lost loved ones can find support, share experiences, and connect with others who understand their journey.

## Getting Started

### 1. Install Dependencies

Open your terminal in the project folder and run:

```bash
npm install
```

### 2. Run the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Current Features

- **Beautiful Landing Page**: Welcoming design with clear messaging about the platform's purpose
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern UI**: Built with Tailwind CSS for a clean, professional look
